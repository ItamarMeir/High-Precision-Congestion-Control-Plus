Run an HPCC or HPCC+ research workflow in this repository.

Requirements:
- Start from the user's hypothesis or desired improvement.
- Use `HPCC_ALGORITHM_DETAILED.md` and `HPCC_PLUS_README.md` as the algorithm contract.
- Use the `brain`, `coder`, and `verificator` skills through the `hpcc-orchestrator` skill.
- If code changes are needed, implement and verify them before claiming success.
- If a simulation config is provided and execution is feasible, use the repository helper scripts and summarize the resulting plots.
- If simulation is skipped, state exactly what remains unvalidated.

Expected output:
- Research plan
- Implementation status
- Verification findings or approval
- Simulation and plotting summary if executed
