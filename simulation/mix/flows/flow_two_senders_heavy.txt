Flow Format Notes:
Write size is in BYTES (passed to RdmaClient `WriteSize`).
- src, dst: source and destination node IDs (hosts only, not switches)
- 3: protocol (fixed)
- dport: destination port number
- write_size_bytes: total bytes to send
  PACKET_PAYLOAD_SIZE is the payload size for each packet (default: 1000 bytes, set in config)
- start_time: flow start time in seconds

Example (1Gbps links, congestion to ~500Mbps per flow):
  write_size_bytes=100,000,000 bytes = 100 MB
  Expected FCT: 100 MB ÷ 500 Mbps = 1.6 seconds

Please order the flows in ascending start_time!
